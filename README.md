# Hotel Room Booking App

A user-friendly web application to browse, book, and manage hotel rooms.  
Supports multiple room types, pricing, capacities, and extra facilities.  
Includes an admin panel to add, update, and delete rooms with images.  
Responsive design for seamless experience on desktop and mobile.  
Built with modern web technologies for fast and secure bookings.
Link:https://hotel-booking-rooms-beach-resort.vercel.app/
